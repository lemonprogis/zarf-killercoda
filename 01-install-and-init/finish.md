# You're Zarf-ready

You've installed the `zarf` CLI and bootstrapped a Kubernetes cluster with the init package.
Your cluster now has the registry, agent, and tooling needed to deploy Zarf packages.

## What's next

- **Deploy the Retro Arcade** — try `zarf package deploy` against a real package
- **Create Your First Package** — build your own `zarf.yaml` and `zarf package create`
